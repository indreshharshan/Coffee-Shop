# ☕ Bean & Brew Coffee Shop

Welcome to the **Bean & Brew Coffee Shop** website — a beautiful and responsive static site built using **HTML & CSS**!  
This project was created as part of my **12 Weeks 12 Projects Challenge**.

🔗 Live Demo: https://beanbrewcoffeeshop.netlify.app

---

## 🌟 Project Overview

Bean & Brew is a coffee shop landing page that showcases:

✔️ Hero section with a welcoming header  
✔️ About Us section describing the coffee experience  
✔️ Menu section with various drinks  
✔️ “Why Us” highlights shop features  
✔️ Gallery of images and ambiance  
✔️ Customer testimonials  
✔️ Contact form and contact info  
✔️ Fully styled with **HTML & CSS**

This site is fully responsive and works well on desktop and mobile devices.

---

## 📁 Features

- Clean layout with semantic HTML structure  
- Attractive styling using CSS (no frameworks)  
- Menu and pricing showcase  
- Testimonials for user feedback  
- Contact section with form  
- Gallery to display images  
- Smooth navigation
